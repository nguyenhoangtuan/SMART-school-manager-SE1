System Requirements

To use QuickManage 2013, your system must meet those requirements:
  • CPU: Pentium 3 or higher
  • RAM: 1GB or more
  • HDD: 1GB free space or more
  • OS: Windows XP/7 or Linux Fedora 17/18 or Linux Ubuntu 12.04/12.10 with appropriate JRE installed.
  • JRE: 1.7 or higher

    •

    Running Enviroment

    If your system has no JRE installed yet, JRE can be downloaded     here.
    Clean your hard disk to have at least 1GB MB free space. Additional free space may be required to store database and other user data.
    JRE installation instructions can be found here.

    **Note: Minimum JRE version required is 7.

    Install QuickManage 2013

    QuickManage 2013 is a Java-based desktop application, so all you need to do is to download or copy the .jar file to your desire destination. Then double click on it.
    On most systems with appropriate installed JRE, the program will run automatically.
    If it does not run automatically, or is opened by an archive manager, please run the following command in your CMD:
    java -jar [Jar file Name]
