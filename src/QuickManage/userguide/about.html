QuickManage 2013 is a Java desktop program that manage art schools in Ho Chi Minh City.
This program stored classes, students, teachers and class schedule for art schools. This program provides user with intuitive user interface, effective management tool and secure data storage.

This program can be managed by both admins and staffs with appropriate privilege.
