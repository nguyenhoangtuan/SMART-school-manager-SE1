How To Use

Overview

When you first startup the program, login windows appears:



After provided username and password, user click on to login.

Admin

To use this program as a Admin, you should acquire activated account with admin right

Admin accounts have Admin Functions (Account Management) and Shared Functions (Teacher, Class Management etc which shared with Staff accounts)

This is the main window for Admin:



The program list all user account on account tab (default), dividing into xx accounts each pages. Users can click on the page number buttons located at the middle bottom line to switch back and forth.

Admin Functions: Account Management

Add Account:
To Add Account, admins click on , account information form appears as below:



Information required:
  • Name: Account name
  • Password: Account password
  • Email: Account holder's email as format: emailaccount@domainname.suffix for example binhnhuonghau@gmail.com
  • Phone number: Account holder's phone number as format: xxx-xxx-xxxx (x is number) for example 091-111-1111
  • Role: Account privilege level (Admin or Staff)
  • Image: Optional, staff or admin portrait can be posted here

    • Edit Account:
    To Edit Account, admins click on , account information form appears as below:



    Information required:
  • Name: Account name
  • Password: Account password
  • Email: Account holder's email as format: emailaccount@domainname.suffix for example binhnhuonghau@gmail.com
  • Phone number: Account holder's phone number as format: xxx-xxx-xxxx (x is number) for example 091-111-1111
  • Role: Account privilege level (Admin or Staff)
  • Image: Optional, staff or admin portrait can be posted here

    • Delete Account:
    To Delete Account, admins choose account for deletion then click on

    Be aware that after deleted, account will be deleted permanently and action can not be undo.
    Delete All Account:
    To Delete All Account, admins click on

    Be aware that after deleted, all accounts except for default admin account will be deleted permanently and action can not be undo.

    Deactivate Account:
    To Deactivate Account, admins choose account for deactivation then click on then account status will be changed to "false" as below:



    After deactivation, account can be re-activated at any time. Login will not be possible until account activated again.

    Activate Account:
    To Activate Account, admins choose account for activation then click on then account status will be changed to "true" as below:



    After activation, account can be de-activated at any time. Login will be possible after the action.
